## Descripción del Proyecto

Sistema de gestión de relaciones con clientes (CRM) desarrollado con tecnologías web frontend. La aplicación permite administrar información de clientes y mantener un registro de notas asociadas a cada uno de ellos.

## Arquitectura Técnica

### Frontend
- **HTML:** Estructura y marcado de la aplicación
- **JavaScript:** Lógica de negocio y manipulación del DOM
- **Tecnología:** Single Page Application (SPA)

### Despliegue
- **Plataforma:** Vercel
- **Tipo:** Aplicación estática
- **CI/CD:** Integración automática desde repositorio GitHub

## Funcionalidades Principales

### Gestión de Clientes
- Registro y almacenamiento de información de clientes
- Visualización de listado de clientes
- Edición y actualización de datos de clientes
- Eliminación de registros de clientes
