# CRM App - Gestión de Clientes y Notas

Una aplicación web simple y eficiente para la gestión de clientes y sus notas asociadas, desarrollada con tecnologías web nativas.

## 🚀 Demo en Vivo

[Ver aplicación en funcionamiento](https://app-fcgtk-q2u8p6oam-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- ✅ Gestión completa de clientes (CRUD)
- 📝 Sistema de notas por cliente
- 💾 Almacenamiento local persistente
- 📱 Interfaz responsive
- 🔍 Búsqueda y filtrado de clientes
- 🎨 Interfaz intuitiva y moderna
- ⚡ Carga rápida sin dependencias externas

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Almacenamiento**: LocalStorage
- **Hosting**: Vercel
